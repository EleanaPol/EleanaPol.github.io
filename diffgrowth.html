Differential Growth Objects

December 2019 - Ongoing

This is an ongoing projects that explores the manufacturing of objects generated from differential growth.






­

­






Project Description

Design

As part of this exploration, two different objects have been designed and fabricated. The first, completed in 2019, is a mug engraved with a differential line, and produced as a ceramic 3D print. While in a production, there was a lot of back and forth in order to be over the minimum thresholds of feasibility for this particular manufacturing technique, which ended up in a chunkier object overall.

The second object is a necklace, that has been 3D printed in castable resin and then cast with brass. Due to the nature of the object and the need to divide the differential growth line into smaller chunks (beads) it has been a particular challenging process to remain within the fabrication threshods while still being able to produce an elegant jewel. This process is still ongoing.
